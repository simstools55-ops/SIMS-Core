# Output Specification

## 通常出力

依頼内容に応じて、次から必要なものだけを明確な見出しで出力します。

- 改善方針
- 新しい記事タイトル/H1
- 新しいSEOタイトル
- 新しいメタディスクリプション
- 差し替え用導入文
- 推奨見出し構成
- 差し替え・追記本文
- FAQ
- 内部リンク提案
- 変更点の要約

毎回すべてを無理に出力せず、依頼された改善範囲に合わせます。

## JSON

依頼文で指定されたSIMS_FEEDBACK_V1 version 1.1を回答末尾に1つ出力します。JSONの後には何も出力しません。

`expected_effect` は保証値ではありません。根拠が薄い場合は空文字を使います。
